Its a game where you have to press Space to jump, just like Flappybird, but with cats I wanted to make the I'm choosing a character from several cats, but I don't have time, since Onekey is ending soon 🫪 😸

To create this game I first thought of doing like the Google dino game, but then i thought of doing like a Flappy bird, which was way easier to create, but I had some problems with the jump. 

How to Play

Just press the Space to do mini jumps in order to avoid hitting the tubes 😸, the Cat is easly changed, you only have to replace the image from the Cat file, and if you don't want to change the code, just delete the Cat and put yours and rename it to cat13, if you do want to change it is also easy 😉.

It doesn't work with phone, if you want it to be playable on the phone, add a button or add touchable screen!
