# Jumpcat

----------
Spanish ⭐🇪🇸⭐
----------

Es un juego donde tienes que presionar el espacio para saltar, es como en Flappy Bird, pero con gatos. Quería añadir la opción de elegir entre varios gatos, pero no me da tiempo, ya que Onekey va a terminar ya.

Para crear este juego, primero pensé en hacerlo como el juego del dinosaurio de Google, pero luego se me ocurrió hacerlo como Flappy Bird, que era mucho más fácil de crear, aunque tuve algunos problemas con el salto. 

Cómo Jugar

Solo presiona el espacio para dar saltitios y evitar chocar contra los tubos 😸. El gato se cambia fácilmente, solo tienes que reemplazar la imagen en la carpeta cat. Si no quieres modificar el código, simplemente borra el gato actual, pon el tuyo y cámbiale el nombre a cat13. Si prefieres cambiar el código, también es muy fácil 😉.

No funciona en móvil. Si quieres que se pueda jugar en móvil,añade un botón o haz que la pantalla sea táctil

Espero que lo disfrutes ❤️

Its a game where you have to press Space to jump, just like Flappybird, but with cats I wanted to make the I'm choosing a character from several cats, but I don't have time, since Onekey is ending soon 🫪 😸

To create this game I first thought of doing like the Google dino game, but then i thought of doing like a Flappy bird, which was way easier to create, but I had some problems with the jump. 

How to Play

Just press the Space to do mini jumps in order to avoid hitting the tubes 😸, the Cat is easily changed, you only have to replace the image from the Cat file, and if you don't want to change the code, just delete the Cat and put yours and rename it to cat13, if you do want to change it is also easy 😉.

It doesn't work with phone, if you want it to be playable on the phone, add a button or add touchable screen!
